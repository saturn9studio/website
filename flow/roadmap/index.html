Flow Roadmap

Flow is in preview. While the core features to enable creative writing are ready, a lot more features are coming soon.

Markdown

Add support for additional markdown features:

  • Images - ability to insert images.
  • Tables - ability to create and edit tables.
  • Footnotes - ability to create footnotes.
  • Hyperlinks - ability to add hyperlinks.
  • Code blocks - create code blocks with syntax highlighting.
  • Mathematical notation - complex math notation.

Editor

Core editor features:

  • Find and replace - find and replace text in the document.
  • Recents - keep track of recently opened documents.
  • Print - support for printing.

Look and feel

An even more delightful user experience:

  • Additional themes - more themes, both light and dark.
  • Typography options - have multiple fonts to choose from.

Export

Export your work:

  • PDF export - export as PDF document.
  • Word export - export as .docx Word document.
  • HTML export - export as stand-alone .HTML document.

Platform support

Flow will be available on multiple platforms:

  • Apple Store - published on the Apple App Store.
  • Windows Store - published on the Windows Store.
  • iPad - tablet version of Flow.
  • Other platforms under consideration (iPhone, Android)

Inspiration

Features built to spark creativity:

  • Writing prompts - inspirational writing prompts.
  • More to be announced soon.
